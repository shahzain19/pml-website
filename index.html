Questions

Does this work with Cursor? ‚Üì
Yes. PML automatically generates and updates `.cursor/rules.md` based on your project's current state and memory graph, ensuring Cursor always has the latest context.
Is my data sent to the cloud? ‚Üì
No. PML is local-first. The memory graph is stored in your project's `.pml` directory. You own your data.
Does it work with existing projects? ‚Üì
Yes. Run `pml init` in any existing repository. It will scan your file structure and attempt to build an initial memory graph.
